<h1>vue-pure-admin精简版（国际化版本）</h1>
## 介绍
基于 [vue-pure-admin](https://github.com/pure-admin/vue-pure-admin) 
## 用法

### 安装依赖

pnpm install

### 安装一个包

pnpm add 包名

### 卸载一个包

pnpm remove 包名
